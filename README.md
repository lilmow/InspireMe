#  InspireMe Quote Generator

The Quote Generator is a simple web application that allows users to get random quotes from various authors. It is designed to provide a moment of inspiration or reflection by displaying thought-provoking quotes with the click of a button.

## Features

- Click the "New Quote" button to retrieve a new random quote.
- Display of the quote's content and author.
- Attractive and responsive user interface for a seamless user experience.
- Error handling for quote retrieval to ensure a smooth operation.

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API

## Getting Started

To run this project locally, follow these steps:

1. Clone this repository to your local machine.
2. Open the project's HTML file in a web browser.
3. Click the "New Quote" button to get a random quote.
